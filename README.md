A professional, modern, mobile-responsive website for **VR Physio Rehab** (South India).

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- shadcn-style UI primitives (Button, Card, Input, etc.)
- Framer Motion animations
- Lucide React (icons-ready)
- SEO metadata & OpenGraph

## Quick Start
```bash
pnpm i   # or npm i / yarn
pnpm dev # http://localhost:3000
```

- `/blog` + `/blog/[slug]`
- `/privacy`, `/terms`, `/patient-rights`
